# 目的
予想サイトの傾向を分析する

# 情報ソース取得
SELECT [idx]
      ,[開催日]
      ,[開催曜日]
      ,[開催場所]
      ,[R]
      ,[軸]
      ,[相手]
      ,[押さえ]
      ,[ソース]
      ,[メモ]
      ,[的中]
      ,[狙い]
      ,[結果1着]
      ,[結果2着]
      ,[結果3着]
      ,[人気1着]
      ,[人気2着]
      ,[人気3着]
  FROM [hr01].[dbo].[prediction_and_result]
  ORDER BY
    [idx]

# 情報ソース項目
IDX：行番号
開催日：開催日
開催曜日：開催日の曜日
開催場所：競馬場
R：レース
軸：本命、軸馬
相手：対抗馬
押さえ：穴、惑星
ソース：予想サイト
的中：予想結果◎○的中、△外れ(縦目)、×外れ
狙い：券種
結果1着：1着
結果2着：2着
結果3着：3着
人気1着：1着馬人気
人気2着：2着馬人気
人気3着：3着馬人気

# 調査事項
予想サイトの年別、月別、曜日別、レース別：メイン、平場、最終、人気：本命、中荒れ、大荒れ、開催競馬場
の得手、不得手と的中率の傾向を調査して

# 結果出力
解析結果は./解析/予想と結果の傾向.md　に出力して
